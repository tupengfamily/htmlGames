围棋游戏规则 — 新手入门指南

一、什么是围棋？

围棋是起源于中国、已有4000多年历史的策略棋类游戏。双方轮流在棋盘交叉点上落子，通过围地来争夺胜负。围棋规则极其简单，但策略深度无穷，被誉为"人类智慧的巅峰"。

二、棋盘与棋子

  • 棋盘：标准19×19围棋棋盘，共361个交叉点（本游戏使用标准棋盘）
  • 棋子：黑棋和白棋，黑方先行，双方轮流落子
  • 落子：棋子放在交叉点上（不是方格内！），落子后不可移动
  • 星位：棋盘上9个黑点标记，是重要的参考点，通常开局先占角部星位

三、气的概念 (核心！)

"气"是围棋最重要的概念：指一颗棋子在棋盘上相邻的空交叉点（上下左右四个方向，不含斜对角）。

  • 棋盘中央的单颗棋子有4口气
  • 边上的单颗棋子有3口气
  • 角上的单颗棋子有2口气
  • 连接的同色棋子共享气——多颗相邻同色棋子算一个整体
  • 当一组棋子的所有气都被对方棋子堵住时，这组棋子就被"提掉"（从棋盘上移除）

四、提子规则 (吃子)

落下一子后，系统会自动检查对方棋子的气：

  • 如果对方某组棋子的气变为0（被完全围住）→ 对方棋子被提掉
  • 提掉的棋子从棋盘上移除，成为己方的"提子"
  • 每提掉对方一子，相当于多了1目的优势

新手tip：先确保自己的棋子有气，再考虑攻击对方。不要贸然送子！

五、禁着点与打劫

  • 自杀禁手：不能下在没有气且不能提掉对方棋子的位置（走进去自己的棋子立刻无气）
  • 打劫规则 (Ko)：当双方能互相反复提掉对方一子时，不能立即提回，必须先在别处落一子后才能回来提
  • 本游戏系统自动判断禁着点和打劫，若你点了禁止位置会弹出提示

六、胜负判定 (重要！)

本游戏采用国际规则（数子法），计算方式如下：

1. 什么是"目"？

"目"是围棋中计算得失的单位：

  • 棋盘上每个被己方完全包围的空交叉点 = 1目
  • 每提掉对方1颗棋子 = 1目
  • 目数越多越好！

2. 胜负计算公式

  • 黑方总目数 = 黑方围住的空点（地） + 棋盘上黑子数
  • 白方总目数 = 白方围住的空点（地） + 棋盘上白子数 + 贴目6.5目
  • 贴目：因为黑方先手有优势，所以白方额外加6.5目作为补偿（0.5目避免平局）
  • 总目数多者获胜

3. 举例说明

假设终局时：

  • 黑方围了50个空点 + 棋盘上40颗黑子 = 90目
  • 白方围了45个空点 + 棋盘上38颗白子 + 6.5贴目 = 89.5目
  • 黑方90目 > 白方89.5目 → 黑方胜0.5目

新人tip：不用死记公式，只需记住——多围空、多吃子、少被吃，终局比目数！

七、棋局结束

  • 双方停招：双方连续各停一招 (点击Pass按钮)，表示都认为没有有价值的落子点了，棋局结束
  • 认输：任一方可以点击"认输"按钮立即结束棋局
  • 终局后系统自动计算双方目数并宣布胜负

八、新手快速入门策略

  1. 先占角、后占边、再中腹——角落最容易围地，效率最高
  2. 保持棋子连接——连在一起的棋子不容易被吃
  3. 注意气的数量——不要让棋子只剩1口气（叫"被打吃"）
  4. 两眼活棋——一组棋子需要至少两只"眼"（对方无法填满的独立空点）才能确保不被吃
  5. 围空比吃子更重要——初学者容易沉迷吃子，但最终比的是围地面积
